# Personal Portfolio

A clean, responsive personal portfolio website showcasing projects, skills, education, and work experience with smooth animations and modern design.

## Features

- **Responsive Design**: Works seamlessly on all devices
- **Animated UI**: Typing effect and scroll-triggered Framer Motion animations
- **Dark Mode**: Full light/dark theme support with theme aware icons

## Usage

### 1. Clone the Repository

```bash
git clone https://github.com/MianSaadTahir/Portfolio.git
cd Portfolio
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run Development Server

```bash
npm run dev
# or
yarn dev
```

Open `http://localhost:3000` in your browser to see the portfolio.
